Quilt: A python based static site html stitcher.
initial development release (0.1.0)§

by tim on 3/1/2015 3:15 pm

A python based static site html stitcher.

Description
Generate a static site via python from content-only .html and .md pages that are stitched together along with patch (e.g. nav.html, footer.html) files onto a main site quilt.html file.
Code Examples
Quilt a site located at path/to/site/source/files.
from quilt.Quilting import QuiltingRoom

q = QuiltingRoom('path/to/site/source/files')
q.quilt()
